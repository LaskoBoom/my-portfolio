KonPsyk.dk

Search Engine Optimisation & Website Performance

Overview

Client: Anders C. Randløv

Website: www.konpsyk.dk

This project involved improving the search engine optimization (SEO) of the website for KonPsyk, a psychology practice in Odense run by psychologist and director Anders C. Randløv.

The goal was to improvethe site’s visibility in local search results and strengthen its technical SEO foundation. The work included on-page SEO improvements, structured data implementation, performance optimization, and site structure enhancements.

The Challenge

The KonPsyk website had several SEO and technical issues, and initially ranked on page six of Google for local searches for psychiatric services in Odense. The challenge was to improve the site’s visibility by optimizing content, structure and performance so potential clients could easily find the practice online.

The Solution

On-Page SEO

  • Added unique SEO titles and meta descriptions for all pages
  • Optimized content with local and relevant keywords (e.g., “psykolog Odense”)
  • Improved readability and flow to make content more user- and search-friendly

Structured Data & Schema

  • Implemented LocalBusiness schema to strengthen local search visibility
  • Added FAQ schema for rich search results and increased click-through rates

Technical SEO & Performance

  • Standardized heading structure (H1–H3 hierarchy) across all pages
  • Repaired broken links and optimized internal linking
  • Resized large images and installed performance plugins (WP-Optimize, Smush, Asset CleanUp) to improve page speed

Site Structure & Navigation

  • Reorganized page hierarchy into logical parent/child relationships
  • Added breadcrumb navigation to improve user experience and search indexing
  • Removed duplicate and unnecessary pages for cleaner navigation

Accessibility & Usability

  • Added descriptive alt text for images
  • Improved text contrast and visibility of buttons

Technologies Used

WordPress

Outcome

After implementing the SEO improvements, the KonPsyk website consistently ranked on page two of Google for local searches of psychiatry practices in Odense and occasionally reached page one, confirmed via incognito searches and a search engine simulator to avoid browser bias. The site now loads faster, has a cleaner structure, and provides a better experience for visitors, making it easier for potential clients to find and navigate the practice online.
